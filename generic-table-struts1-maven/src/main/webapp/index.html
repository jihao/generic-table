Uniform Table based on Struts 1.3

Do you like the following table with Ericsson look and feel (old), with feature sort, filter, paging ?

Can you believe you can have such table within 30 minutes with several lines of java code ?

Let's start

so, we assume you are in a Java dynamic web project which was based on Struts, and you are going to implement the above list

  • download the generic-table.zip, unzip the file, you'll get
    • 	- generic-table.jar
	- resources/common/uniform_table_list.jsp
	+ resources/css/*
	+ resources/images/*
	+ resources/js/*
	+ lib/* (the 3pp lib dependences)
  • copy the generic-table.jar to you WebRoot/lib directory (if use maven, you need update the pom)
  • copy the resources/* to you WebRoot directory
  • write your ExampleAction which extends com.ericsson.dcp.generic.table.action.UniformTableAction, and implement the following 4 methods

    • You can check the example com.ericsson.dcp.generic.table.example.UserListAction from the source code

		// TODO Auto-generated method stub
		return null;
	}

		// TODO Auto-generated method stub
		return null;

			throws Exception {
		// TODO Auto-generated method stub

  • write the example_list.jsp code to include the uniform_table_list.jsp
    • <head>
	<link href="css/common.css" rel="stylesheet" type="text/css"></link>
	<link href="css/pandora.css" rel="stylesheet" type="text/css"></link>
	<script type="text/javascript" src="js/jquery-1.7.2.min.js"></script>
<head>
...
...
<jsp:include page="common/uniform_table_list.jsp"></jsp:include>
  • configure your struts-config.xml
    • 	
<form-beans>
	<form-bean name="uniformTableForm" type="com.ericsson.dcp.generic.table.action.UniformTableForm"></form-bean>
</form-beans>
<action-mappings>
	<action path="/ExampleAction" name="uniformTableForm" scope="request" type="com.ericsson.***.ExampleAction" parameter="method">
		<forward name="list" path="/example_list.jsp"></forward>
	</action>
</action-mappings>
  • finished, you can start your web server to see it live
